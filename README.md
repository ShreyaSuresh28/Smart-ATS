# Smart ATS 2.0

An AI-powered Applicant Tracking System built using Streamlit.

## Features

- Candidate Signup/Login
- Resume Upload
- Resume Parsing
- Skill Extraction
- ATS Matching
- Candidate Ranking
- HR Dashboard
- Similarity Analysis
- PDF Feedback Reports
- Email Notifications
- Admin Dashboard

## Technologies

- Python
- Streamlit
- SQLite
- NLTK
- Scikit-Learn
- PyPDF2
- ReportLab
- Matplotlib

## Run Project

pip install -r requirements.txt

streamlit run app.py
